Grafico

Se realiza una conexión con la API de Pokémon y se utilizan filtros AJAX para realizar búsquedas dinámicas. El proyecto fue desarrollado con Django, gestionando eficientemente las operaciones CRUD e integrando un sistema de autenticación con superusuario para la administración de usuarios

La solución está enfocada en mejorar la experiencia de usuario, proporcionando una página sin recarga, y separando el código reutilizable en diferentes carpetas para una mejor organización.
